# kodumaro-clock

![screenshot](https://github.com/cacilhas/clock/blob/master/screenshot.png)

Yet another [X11](https://en.wikipedia.org/wiki/X_Window_System) clock.

## Installation

```sh
cargo install kodumaro-clock
```
